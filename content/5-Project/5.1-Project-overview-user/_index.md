#### 5.1.2 Description of Figure 5.2

### 1. Analysis of Existing Features

#### Navigation Area (Sidebar & Header)

- The left sidebar contains all essential modules of an E-learning ecosystem, including **Online Exams**, **Exam Results**, **Classes**, **Question Bank**, **Learning Materials**, and **Notifications**. Each module is accompanied by intuitive icons, making navigation easy for users.

- The header provides quick-access utilities, including a real-time clock, Light/Dark mode toggle, notification center, and user profile management.

#### Quick Exam Access

- The **Join Exam** feature allows students to enter an exam code (for example, **AURA25**) and immediately access an examination.
- This significantly improves the user experience by reducing the number of navigation steps required to join an exam.

#### Statistics Cards

The dashboard contains four summary cards:

- Average Score
- Submitted Exams
- Current Classes
- Practice Sessions

These cards provide students with an instant overview of their academic performance immediately after logging in.

Each card uses a different color (Blue, Green, Yellow, and Pink) to clearly distinguish different categories of information.

#### Detailed Data Sections

**Recent Exam Results**

Displays important examination information in a structured table, including:

- Exam Name / Code
- Date
- Status
- Score

The **"View All"** button allows students to access their complete exam history.

**Learning Overview & Recent Practice**

These sections help students monitor their learning progress, completion rate, and recent practice activities.
