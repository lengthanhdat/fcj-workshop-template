# Nhật Ký Thực Tập Tuần 8

**Thời gian:** 08/06/2026 - 12/06/2026

Báo cáo tiến độ và chi tiết các công việc đã thực hiện trong tuần 8 của kỳ thực tập tại AWS (Dự án Aura Academic).

### Chi Tiết Công Việc

| STT | Nội Dung Công Việc | Ngày Thực Hiện | Tình Trạng | Tài Liệu Tham Khảo |
| --- | --- | --- | --- | --- |
| 1 | Containerization với Docker: Nghiên cứu và viết file cấu hình Dockerfile, .dockerignore cho các dự án Backend (Node.js/TypeScript hoặc Java Spring Boot). Tiến hành đóng gói toàn bộ mã nguồn, biến môi trường và các thư viện AWS SDK thành các Docker Image độc lập. | 08/06/2026 | ✅ Hoàn thành | [Tài liệu AWS](https://cloudjourney.awsstudygroup.com/) |
| 2 | Tối ưu hóa Docker Image: Thực hành kỹ thuật Multi-stage build nhằm giảm tối đa dung lượng Image, giúp quá trình deploy lên Cloud diễn ra nhanh chóng và tiết kiệm tài nguyên lưu trữ. | 09/06/2026 | ✅ Hoàn thành | [Tài liệu AWS](https://cloudjourney.awsstudygroup.com/) |
| 3 | Làm quen với Amazon ECR (Elastic Container Registry): Tạo các kho lưu trữ Private trên ECR, thực hiện cấu hình AWS CLI ở máy Local để đăng nhập (authenticate) và đẩy (push) các Docker Image của dự án lên Cloud an toàn. | 10/06/2026 | ✅ Hoàn thành | [Tài liệu AWS](https://cloudjourney.awsstudygroup.com/) |
| 4 | Nghiên cứu về Amazon ECS (Elastic Container Service) & AWS Fargate: Tìm hiểu giải pháp chạy container dạng Serverless (Fargate) để không phải tự quản lý máy chủ EC2 gốc. Định nghĩa các Task Definitions (cấu hình RAM, CPU, Port) cho container Backend. | 11/06/2026 | ✅ Hoàn thành | [Tài liệu AWS](https://cloudjourney.awsstudygroup.com/) |
| 5 | Thử nghiệm cấu hình mạng và bảo mật cho Container: Thiết lập các luồng ánh xạ cổng (Port mapping) và cấu hình lại Security Group để chuẩn bị cho việc kết nối giữa Container Backend với Database RDS và API Bedrock trên môi trường Cloud. | 12/06/2026 | ✅ Hoàn thành | [Tài liệu AWS](https://cloudjourney.awsstudygroup.com/) |
